# 🌍 Hydrology Groundwater Quality Prediction

AI-powered groundwater quality prediction system using Machine Learning and GIS Visualization to analyze groundwater parameters, predict water quality levels, compare ML models, and visualize results geographically.

## 📌 Project Overview

This project predicts groundwater quality using Machine Learning techniques and provides geographic visualization for better analysis and decision-making.

Users can:
- Upload groundwater datasets
- Automatically clean and preprocess data
- Train and compare ML models
- View prediction results
- Analyze water quality visually using GIS maps
